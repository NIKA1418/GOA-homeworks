რესპონსიული დიზაინი არის მიდგომა, რომელიც უზრუნველყოფს ვებსაიტის სწორად ფუნქციონირებას და ვიზუალურ კომფორტს სხვადასხვა მოწყობილობაზე (მაგალითად: ტელეფონი, ტაბლეტი, კომპიუტერი). მთავარი მიზანია, რომ ვებსაიტი იყოს ადაპტირებული ეკრანის სიგანესა და გარჩევადობას. რესპონსიულობის მიღწევის გზები: - CSS Media Queries: განსაზღვრეთ სტილის წესები კონკრეტული ეკრანის ზომებისთვის. - პროცენტული ზომები: გამოიყენეთ ზომები (%), რათა ელემენტები ავტომატურად ადაპტირდნენ. - Flexbox და CSS Grid: თანამედროვე ლაიაუტების დამუშავება მოქნილი ელემენტებისთვის. - სურათების ადაპტირება: გამოიყენეთ max-width: 100%; რათა სურათები არ გასცდეს მათი კონტეინერის ზომას. - რეაგირებადი ტიპოგრაფია: ზომების გამოყენება em ან rem ერთეულებით. ვიღებდით რაც შეიძლება ინტუიციურად: თექვსმეტობითი სისტემა (HEX) იყენებს 16-სიმბოლურ სისტემას, რომლისთვისაც გამოიყენება ციფრები 0-დან 9-მდე და ასოები A-დან F-მდე.
